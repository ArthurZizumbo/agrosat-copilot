## EDA
Para ejecutar el notebook se necesita crear una carpeta llamada Data y dentro descomprimir el dataset de PASTIS-R, el cual se puede encontrar en: https://www.eotdl.com/datasets/PASTIS-R

El folder deberá verse asi:
- data
  - PASTIS-R
     - ANNOTATIONS
     - DATA_S1A
     - DATA_S1D
     - DATA_S2
     - INSTANCE_ANNOTATIONS
     - (otros archivos sueltos)

Ejecutar instalar todas las dependencias de requirements.txt y ejecutar el notebook.
