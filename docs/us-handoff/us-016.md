### Bug-6 detectado y corregido — ERA5 sub-pixel + property name

**Archivo**: `ml/ingest/gee_sampler.py` función `sample_era5_monthly_climate`.

**Síntoma**: en el smoke real con parcelas del fixture demo (~1 km² cada una), las 24 columnas ERA5 mensuales venían en `None`. El payload de `reduceRegions(scale=11132)` solo contenía `{parcel_id}`, sin la propiedad reducida.

**Root cause** (dos issues compuestos):
1. `scale=11132` (nativo ERA5-Land 11 km/píxel) con geometrías sub-píxel → `reduceRegions` no intersecta píxel y omite la propiedad por completo (silenciosamente).
2. Cuando sí hay propiedad, GEE renombra al nombre del band original (`temperature_2m`, `total_precipitation_sum`) en lugar de `mean` cuando se usa `reduceRegions` sobre single-band images. El código leía `props["mean"]` → `None` aun con payload presente.

**Fix aplicado**:
- `scale=11132` → `scale=1000` (oversampling 11x) + `tileScale=4` (evita memory errors en parcelas grandes).
- `props["mean"]` → `props.get(band_name, props.get("mean"))` con fallback (mantiene compat con mocks viejos y con paths donde el scale nativo SÍ produce "mean").
- Nuevo test de regresión `test_era5_reads_band_name_property_not_only_mean` que valida el path real (mock devuelve solo `temperature_2m`/`total_precipitation_sum`, sin `mean`).

**Validación post-fix** (smoke real GEE, Pianura Padana 2024):
- enero `tmean = 3.19 °C` (rango histórico Po Valley 2-4 °C ✓)
- julio `tmean = 24.27 °C` (rango histórico Po Valley 24-26 °C ✓)
- AC-6 (`jan < jul` hemisferio norte): ✅ True
- precipitación anual `~1764 mm` (Po Valley típico 800-1200 mm; el oversampling `scale=1000` puede inflar ligeramente la suma — aceptable para feature engineering, los stats relativos entre parcelas siguen siendo válidos para downstream ML US-019/020). Se anota como **Obs-Bug-6.1** para revisitar calibración exacta en US-019 baseline si el valor afecta importance feature ERA5.

**Tests**: 10 passed + 3 skipped en `test_s1_srtm_era5_samplers.py` (antes 9+3). Suite completa: **140 passed + 3 skipped**, sin regresiones.
