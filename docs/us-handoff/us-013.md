### Decisiones tecnicas no obvias

- **`pytest.importorskip("streamlit")`** + **`pytest.importorskip("streamlit.testing.v1")`** en `tests/app/test_dashboard.py`: tests se activan automaticamente cuando se instale el grupo `paper`, sin marcas manuales.
- **`AppTest.from_function(render_tab_X)`** en lugar de `AppTest.from_file` para los tabs individuales: evita re-cargar todo el dashboard por cada test.
- **`@st.cache_data(show_spinner=False)`** para `load_csv` y `load_yaml`; **`@st.cache_resource`** para `build_folium_map`.
- **Cap defensivo folium** `_FOLIUM_TOP_TILES = 10` (dissolve por TILE en PASTIS) evita renderizar 2,433 poligonos.
- **Cap defensivo `st.dataframe`** `_DATAFRAME_HEAD_ROWS = 200`.
- **Fallback graceful sin streamlit-folium**: `st.info("streamlit-folium no instalado...")` en lugar de crash.
- **Reproyeccion automatica** a EPSG:4326 si PASTIS metadata viene en EPSG:2154.
- **WeasyPrint ausente**: import diferido dentro de `_html_to_pdf`. Lanza `RuntimeError` con mensaje claro pidiendo `poetry install --with paper`. CLI captura con `typer.Exit(code=1)`.
- **GTK Windows**: `OSError` capturado dentro de `_html_to_pdf` envuelve con guia explicita (URL instalador GTK + sugerencia WSL2).
- **`base_url` weasyprint**: `figures_dir.parent.parent` (raiz repo) para que el HTML resuelva imagenes con paths absolutos relativos.
- **Reproducibilidad PDF**: `_collect_figures` ordena alfabeticamente garantizando que el PDF sea bit-identical entre runs.
- **Seguridad XSS**: `Environment(autoescape=select_autoescape(...))` activo (test valida que `<script>` se escapa).
- **Reemplazo ASCII**: `×` (U+00D7) → `x` para satisfacer ruff RUF001/RUF002 en headings del dashboard.
