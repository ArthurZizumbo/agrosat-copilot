# Manual Test — US-017 / US-016b · Destilacion FarSLIP

> US sin UI, sin backend SSE, sin tools ADK. Los flujos manuales validan
> reproducibilidad del training, smoke del extractor desde GCS y
> materializacion Dagster — todo via CLI / Dagster UI / MLflow UI, no
> via browser de usuario final. No hay chat trilingue ni AOI drawer
> que validar para esta US.

## Pre-requisitos

- Rama `us-017` con working tree de Fase 3 aplicado (o PR mergeado a `develop`)
- `poetry install` ejecutado al menos una vez en el repo raiz
- Para flujos GCP: `GOOGLE_APPLICATION_CREDENTIALS` apuntando a service account
  con permisos `storage.objectViewer` sobre `gs://agrosat-models/farslip/`
- Para flujo training real (opcional): VM GCP `g2-standard-8` con L4 24 GB spot
