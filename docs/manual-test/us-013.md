## Limitaciones conocidas (Fase A)

- **Sin `streamlit` local**: la suite `tests/app/test_dashboard.py` skipea via
  `pytest.importorskip` y la cobertura efectiva queda en 0% hasta instalar
  grupo `paper`.
- **Sin `weasyprint` local**: 1 test del export skipea. La CLI lanza
  `RuntimeError` con mensaje claro al invocar `make eda-pdf`.
- **Windows + weasyprint**: requiere GTK3 runtime. Recomendado WSL2 para
  evitar el setup binario.
- **Figuras vacias**: dashboard degrada graceful con `st.info` placeholder;
  el PDF muestra `<p class="placeholder">Pendiente: no se encontraron
  figuras...</p>` en lugar de imagenes.
- **AC-10 (re-run desde dashboard)**: no implementado (opcional-bonus).
- **AC-11 (Playwright smoke)**: placeholder echo (opcional-bonus, planeado
  para Fase B con Nuxt).
