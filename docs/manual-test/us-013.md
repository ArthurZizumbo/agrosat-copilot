## Limitaciones conocidas

- **Sin `streamlit` local**: tests `tests/app/test_dashboard.py` skipean limpio via `pytest.importorskip`. Cobertura efectiva 0% hasta que se instale grupo `paper`.
- **Sin `weasyprint` local**: 1 test skipea. CLI lanza `RuntimeError` con mensaje claro al invocar.
- **Windows + weasyprint**: requiere GTK3 runtime. Recomendado WSL2 para evitar el setup binario.
- **Figuras vacias**: el dashboard degrada graceful con `st.info` placeholder. El PDF mostraria secciones sin imagenes (no es ideal — esperar a que US-010/011/012 corran end-to-end antes de generar PDF final).
- **AC-10 (re-run desde dashboard)**: NO implementado por ser opcional-bonus.
- **AC-11 (Playwright smoke)**: placeholder echo. NO implementado.
