## Resumen ejecutivo

US-017 entrega la **rama de destilacion CLIP parche-a-parche** que produce embeddings espaciales fine-grained de 512-dim sobre crops Sentinel-2 256×256 px (4 bandas B02/B03/B04/B08) de las tres regiones italianas (Pianura Padana, Toscana, Puglia). Las features destiladas se anexan a la Familia 1 del banco de embeddings (junto con AlphaEarth 64-dim semantico global y DINOv3 estructura vegetal) y alimentan el cabezal open-vocabulary del SegFormer-B2 de US-025.

La implementacion sigue el paper FarSLIP §3.2-§3.3:
- **Teacher**: `openai/clip-vit-base-patch16` (3 canales RGB puros, frozen).
- **Student**: copia del teacher con `patch_embed.proj` adaptado a 4 canales (init Kaiming sobre NIR), entrenado con `L_total = α·L_patch_distill + β·L_cls_align + γ·L_contrastive` (α=1.0, β=0.5, γ=0.2).
- **Forward teacher** recibe `images[:, :3, :, :]` (slicing B04/B03/B02 -> RGB) — preserva la semantica AC-2 sin contaminar al teacher con una proyeccion NIR no entrenada (fix Q1 HIGH).
- **Prototipos textuales** calculados 1× por epoch (text encoder frozen, evita re-compute y non-determinism por dropout activo).

El pipeline esta organizado en 4 capas con SoC estricta:
- **Entrenamiento** ([`ml/farslip/distill.py`](../../ml/farslip/distill.py), [`ml/farslip/train.py`](../../ml/farslip/train.py), [`ml/farslip/dataset.py`](../../ml/farslip/dataset.py), [`ml/farslip/dataset_audit.py`](../../ml/farslip/dataset_audit.py), [`ml/farslip/cap_vocabulary.yaml`](../../ml/farslip/cap_vocabulary.yaml)) — `FarSLIPDistillationTrainer`, dos perdidas (`PatchDistillationLoss`, `RegionCategoryAlignmentLoss`), builder de pares imagen-texto con plantillas it/es/en, CLI Typer + audit gate AC-3 (n_pairs ≥ 30 000 + balance ≥ 0.20).
- **Orquestacion** ([`dagster_project/assets/sentinel2_crops.py`](../../dagster_project/assets/sentinel2_crops.py), [`dagster_project/assets/farslip.py`](../../dagster_project/assets/farslip.py)) — dos assets con lineage limpio `sentinel2_crops_256 -> farslip_embeddings_italy`, particionados por (roi, year) leyendo `manifest["crop_year"]` real (fix Q4 HIGH).
- **Serving / extractor downstream** ([`ml/extractors/farslip_extractor.py`](../../ml/extractors/farslip_extractor.py)) — `FarSLIPExtractor` con API publica `extract_embeddings(crops) -> (B, 512)`, `extract_patch_features(crops) -> (B, 196, 768)`, `encode_text(prompts)`, `load_crops_batch(paths)`. Cache local `~/.cache/agrosat/farslip/` con lock + retry hasta 3× + checksum SHA1 + cleanup defensivo en `BaseException` mid-download (fix MT-3).
- **Utilidades DRY centralizadas** ([`ml/utils/gcs_errors.py`](../../ml/utils/gcs_errors.py), [`ml/utils/git_meta.py`](../../ml/utils/git_meta.py), [`ml/utils/seed.py`](../../ml/utils/seed.py), [`ml/utils/geo_io.py`](../../ml/utils/geo_io.py)) — `is_gcs_auth_error()` tipando google.auth.exceptions + google.api_core.exceptions explicitas (Q3+MT-2), `git_sha()` + `dvc_data_version()` reemplazando 3 duplicados (Q5), `propagate_seed()` con `torch.use_deterministic_algorithms(True)` + `CUBLAS_WORKSPACE_CONFIG` (Q6), `write_crop_tiff/read_crop_tiff` (Q10).

## Seguridad y performance

**Seguridad**:
- `yaml.safe_load` confirmado (sin `yaml.load(Loader=Loader)`).
- `torch.load(path, map_location=..., weights_only=True)` aplicado en `train.py:90` — bloquea deserializacion pickle arbitrario en futuros `--resume` desde GCS multi-tenant.
- Subprocess `git_meta.py` usa `shell=False` implicito + argv en list form + `shutil.which()` para resolver `git_bin` — sin input usuario, `# noqa: S603` justificado.
- `_is_gcs_auth_error()` (en `ml/utils/gcs_errors.py`) importa `google.auth.exceptions.{DefaultCredentialsError, GoogleAuthError, RefreshError}` + `google.api_core.exceptions.{Forbidden, NotFound, PermissionDenied, Unauthenticated}` explicitas. Fallback por nombre solo si las libs no estan instaladas. `AttributeError`, `KeyError`, `ValueError` reales burbujean (no se tragan).
- Cleanup defensivo MT-3 en `_download_from_gcs`: `try/except BaseException` -> `dest.unlink()` (con `suppress(OSError)`) -> `raise`. Evita cache parcial leido como valido en la siguiente invocacion. Cubierto por `test_download_failure_leaves_no_partial_cache` con `gs://invalid-bucket` smoke (2 corridas consecutivas EXIT 0).
- Sin secretos hardcodeados, sin `eval`/`exec`/`os.system`/`shell=True`, sin `pickle.load` directo.

**Performance**:
- **Acumulador embeddings (Q9 fix)**: refactor de `list[dict]` Python (~60 MB para 30k pares) a `embeddings_buffer: list[np.ndarray]` por batch (1 alloc CPU por batch, no por fila) + `np.concatenate` final + `pl.DataFrame` columnar. Reduce overhead Python por fila ~10× para 30k pares.
- **Prototipos textuales** calculados 1× por epoch (text encoder frozen) — evita re-compute innecesario y non-determinism por dropout activo.
- **Cache local** `~/.cache/agrosat/farslip/` con lock + retry 3× + checksum SHA1 — evita re-descargas de pesos (170 MB) por proceso.
- **Tests determinism (Q8 fix)**: `test_two_steps_same_seed_bit_exact_state_dict` reescrito con fixture `trainer` scope-module (1× descarga CLIP) + reseed + `student.load_state_dict(init_state)` entre runs. Tiempo: ~2 s tras carga inicial, entra en `make check` sin marca `slow`.
